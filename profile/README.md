## 关于我们
XtremeWave是一支专注于游戏开发和技术创新的团队，致力于为玩家提供高质量的游戏体验。<br>
我们在GitHub上有多个开源项目，例如TownOfNewEpic_Xtreme。<br>
XtremeWave使用现代技术栈如TypeScript和VuePress来开发和维护他们的网站和服务。
